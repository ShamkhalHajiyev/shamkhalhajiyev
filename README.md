#### **Customer Churn Prediction**
Developed a retention model to identify at-risk users, enabling targeted intervention campaigns. Built using Python and scikit-learn, the model analyzes user behavior patterns and engagement metrics to predict churn probability. Deployed on Azure ML for scalable inference, the system helps reduce customer attrition through data-driven retention strategies.

**Tech Stack:** Python, scikit-learn, Azure ML

#### **Order Conversion System**
Designed a real-time predictive engine to identify high-value sessions and optimize conversion rates. Integrated SHAP values for granular explainability, providing business stakeholders with transparent insights into model decisions. The system processes live user interactions and delivers instant predictions to drive targeted marketing actions.

**Tech Stack:** Azure ML, CatBoost, Power BI

#### **Forecasting Engine**
Built a hybrid forecasting stack combining Deep Learning (LSTM) and statistical methods (ARIMA) for robust demand planning. The ensemble approach leverages the strengths of both neural networks and time-series analysis to deliver accurate predictions across varying market conditions. Deployed in production to support inventory optimization and resource allocation.

**Tech Stack:** Python, TensorFlow, Azure ML

#### **ETL & Metrics Automation**
Engineered a high-speed data pipeline for automated reporting, significantly reducing manual latency and improving data freshness. The system orchestrates complex data transformations using Airflow, processes large-scale datasets with Polars for optimal performance, and delivers real-time metrics to stakeholders.

**Tech Stack:** Airflow, Polars, Python

#### **Risk Analytics Platform**
Developed an interactive dashboard for portfolio risk assessment and visualization. The platform provides real-time risk metrics, scenario analysis, and comprehensive reporting capabilities. Built with Dash for dynamic visualizations, containerized with Docker for consistent deployment, and backed by MySQL for reliable data storage.

**Tech Stack:** Dash, Docker, MySQL
